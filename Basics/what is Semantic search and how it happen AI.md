
  # what is meant By Semantic Search ?
      Semantic search is search technique that understands and intent your query - not just exact keyword matching 


Normal search - keyword matching
Semantic search - meaning matching 

** Example of Keyword matching ** -  In a sql database when we execute SELECT * FROM STUDENTS WHERE NAME = "RAVI"  
 first it goes to name column , look for all "RAVI" then if keyword match data retrieve 

 ** How Is This Different from Semantic Search? **

In semantic search, the system understands the meaning or context of the query.For example, if you search for:
Ravi ,A semantic search system might retrieve:Ravindu,Ravindra,Ravi Kumar,Because it understands relationships or similarities — not just exact string matches.
